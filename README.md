# 🗺️ 夢の地図 (Yumechizu)

> AIが**あなたの夢をプロジェクト化**するWebアプリケーション

## ✨ 特徴

- **🤖 AI 統合**: Llama を活用した自然言語理解
- **📋 タスク管理**: 直感的なタスク追加・管理UI
- **📅 カレンダー機能**: スケジュール可視化
- **🗺️ マインドマップ**: プロジェクト進捗の視覚化
- **📱 PWA 対応**: オフラインでも動作（基本機能）
- **🌐 多言語対応**: 日本語、英語、中国語、韓国語など11言語
- **💾 データ永続化**: LocalStorageで自動保存
- **📤 エクスポート/インポート**: JSON 形式でデータ共有

---

## 🚀 クイックスタート

### 前提条件
- Node.js v20 以上
- npm@9+
- Wrangler v3+（Cloudflare Pages 開発ツール）

### インストールと起動

```bash
# 1. プロジェクトディレクトリに移動
cd /home/arisawa/yumechizu/yumechizu-pages

# 2. ローカルサーバーを起動（推奨）
bash start-dev.sh

# または、直接コマンドで実行
wrangler pages dev --local
```

ブラウザで **[http://localhost:8788](http://localhost:8788)** を開きます。

- **フロントエンド**: Vanilla JavaScript + HTML5 + CSS3
- **ホスティング**: Cloudflare Pages

## 🤖 AI機能について

### エンドポイント
```
POST https://yumechizu-ai.arimitsuru.workers.dev/api/chat
Content-Type: application/json

{
  "messages": [
    {
      "role": "user",
      "content": "あなたの夢やアイデア"
    }
  ]
}
```

### レスポンス
```json
{
  "reflection": "あなたのアイデアの洞察",
  "emotion": "関連する感情",
  "question": "さらに考えを深めるための質問",
  "next_step": "実行に向けたアクション"
}
```

### 制限事項
- 1日5回まで利用可能
- オンラインが必須（PWA オフライン機能では AI は使用不可）

## 📊 データ管理

### 自動保存
- すべてのデータは自動的に LocalStorage に保存
- ブラウザをリロードしてもデータが残ります

### エクスポート
**セクション → 📤 データ転移 → 📥 エクスポート**

JSON ファイルを保存して、以下に活用：
- チームメンバーとの共有
- バックアップ
- 外部ツール（Excel, Notion等）への取り込み

### インポート
**セクション → 📤 データ転移 → 📤 インポート**

以前保存された JSON ファイルを選択して復元

## 🐛 トラブルシューティング

### サーバーが起動しない
```
エラー: "This Worker requires compatibility date..."
対応: wrangler を最新版に更新
$ npm install -g wrangler@latest
```

### AI 機能が動作しない
- ネットワーク接続を確認
- ブラウザのコンソール（F12）でエラーメッセージ確認
- API エンドポイント可用性を確認

### データが保存されない
- シークレットモード（プライベートウィンドウ）では動作しません
- 本来のウィンドウで開く
- ブラウザのキャッシュクリアを試す

詳細は [PROJECT_SIMULATION_GUIDE.md](PROJECT_SIMULATION_GUIDE.md#-トラブルシューティング) を参照

## 🔐 セキュリティとプライバシー

- **データ保存**: すべてのデータは ブラウザの LocalStorage（ローカル）に保存
- **サーバー送信**: AI機能使用時のみ、メッセージコンテンツが外部APIに送信
- **個人情報**: PII（個人識別情報）を入力しないようにしてください
